Knight's Tour Java, Bootstrap

The Knight's Tour is a sequence of moves that permits the Knight to land on each coordinate in a chess board only once. Choose between a strategy of random moves or strategy based on a heuristics model. This program will show you what moves were made in its attempt to land on each of the 64 coordinates only once. Built with Java & Bootstrap, hosted on AWS with an Elastic Beanstalk/EC2 instance. Wikipedia: Knight's Tour

Price Checker PHP

A program built to quickly compare prices of products at Canada Computers to competitiors such as Amazon Canada. Using a barcode scanner, a UPC is entered and looked up on a UPC database site (upcitemdb). The program proceeds to query Canada Computers and Amazon for the model number. Prices for the product are then displayed. Hosted on Sheridan's LAMP (Linux, Apache, MySQL, PHP) server. Currently non-functional due to recent updates to the Canada Computers website.
